FashionFlow

A modern e-commerce demo built with React + Vite + Tailwind (frontend) and FastAPI + MongoDB (backend). Includes JWT auth, products, orders with Stripe, and Cloudinary uploads.

Quick start
- Copy .env.example to both apps and fill values
- npm install (frontend) and pip install -r requirements.txt (backend)
- Start: npm run dev (frontend) and uvicorn main:app --reload (backend)
